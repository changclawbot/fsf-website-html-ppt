## 說明
這組頁面把嵌入式客服 Widget 包成兩個品牌版本：
- PetAngel 毛天使消費者客服
- 富盛豐 OEM / ODM 代工客服

## 關鍵前提
- `data-site="petangel"` 必須綁到毛天使專用知識庫
- `data-site="fsfoem"` 必須綁到富盛豐 OEM 專用知識庫
- 兩條客服線的知識、prompt、回答邊界需持續完全分離
